# 📱 Quotes App (iOS)

A simple and elegant Quotes App built for iOS that delivers daily inspiration through a collection of curated quotes.

---

## ✨ Features

- 📖 Browse random quotes
- 🔄 Generate new quotes instantly
- ❤️ Save favorite quotes
- 📤 Share quotes with others
- 📚 Category-based filtering
- ⚡ Fast and lightweight UI
- 💾 Offline support (if implemented)

---

## 🛠 Tech Stack

- Language: Swift / SwiftUI (or UIKit)
- Architecture: MVVM (recommended)
- Storage: Local JSON / CoreData / UserDefaults
- APIs: (if used) Quotes API
- Tools: Xcode, CocoaPods / Swift Package Manager
